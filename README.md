# ultraytics-easy-config-jetson
#  JETSON 환경에서 스크립트 하나만으로 ultraytics 를 사용할 수 있게 됩니다. 

사용방법 

코드를 생성하고 결과물을 생성할 메인 디렉토리를 구성하여 주세요

메인 디렉토리에 현재 jetpack_find_run.sh 와 ultra_hetpack.sh을 넣어주세요 

터미널로 해당 디렉토리 경로로 가주세요 

터미널 : ./chmod + x ultra_jetpack.sh  jetpack_find_run.sh
터미널 : sudo su 
터미널 : ./ultra_jetpack.sh 


jetpack_find_run.sh 파일은 나중에 docker stop을 할 경우에만 사용 해주시길 바랍니다. 

해당디렉토리 경로 이동후  

터미널: ./jetpack_find_run.sh
